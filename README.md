## 支持的广告格式全面覆盖

PPCmate支持5种主流广告格式，基本涵盖了你能想到的所有需求：

**1. Push通知广告（CPC计费）**
这是我个人最喜欢用的格式。点击率通常在3-8%之间，转化效果也不错。适合做联盟营销、APP推广这类需要快速转化的项目。

**2. Pop-under V2广告（CPM计费）**
就是弹窗广告的升级版。虽然有人觉得这种形式有点侵入性，但对于某些垂直领域（比如约会、游戏）效果确实好。而且PPCmate的反作弊系统做得还行，至少不会给你一堆机器流量。

**3. Display展示广告（CPM计费）**
传统的横幅广告，适合做品牌曝光。PPCmate支持多种尺寸，300×250、728×90这些常见尺寸都有。

**4. Native原生广告（CPM计费）**
这种广告看起来像网站的正常内容，用户接受度比较高。我测试下来CTR能到2-4%，比硬广效果好不少。

**5. Video视频广告（CPM计费）**
视频广告的CPM会贵一些，但对于需要详细展示产品的场景很有用。

每种格式都有详细的定位选项，包括地理位置、设备类型、浏览器语言、兴趣标签等。基本上你想要的精准定位功能都有。
